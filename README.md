# Library Management System

A simple command‑line library system written in Python.  
Allows adding/removing books, registering users and recording borrow/return transactions.

## 📂 Repository

This repo contains:

- `main.py` – program entry point
- `book.py`, `user.py`, `transaction.py`, `database.py` – core modules
- `library.db` – SQLite database file (tracked for simplicity)
- `__pycache__/` – compiled files (ignored via `.gitignore`)
